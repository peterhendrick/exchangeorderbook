
Combined Order Book
=======

This package will provide a full-stack, combined exchange order book web application. Using socket.io, the tables will
live update.

To download:
```bash
git clone https://github.com/peterhendrick/exchangeorderbook
```

To install:
```bash
npm install
```

To test:
```bash
npm test
```

To run:
``` bash
npm start
```
